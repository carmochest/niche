02 — The mark

One punctuation mark, three shapes.

The mark sits where a period would — after the name. It takes the shape of whichever value is speaking. Click a shape to set the page's state.

The blob
Creativity — never the same twice

Hand-drawn, irregular, alive. The shape of an idea before it has rules. Its outline drifts constantly, refusing symmetry.

The circle
Humanity — whole, breathing

The oldest symbol of people gathered. It breathes slowly — in and out — like the room it represents.

The triangle
Entrepreneurship — pointed forward

A vector, not a shape. It leans and tips toward what's next — momentum frozen mid-launch.

It's punctuation, not a logo.

The mark never floats beside the wordmark like a corporate symbol. It ends the name — NICHE● — or stands completely alone as an avatar, favicon, or stamp. Never both at once.

In motion on screens, frozen in print.

Digital surfaces let the mark move and morph between states. Print picks one shape, one color, and commits — the same freeze rule as the wordmark's weight.

10 — The system

Four ingredients, no exceptions.

Print freezes everything

Static media use weight 900 and one mark shape in one color. Motion is a screen privilege, not a requirement.

Weight = attention, shape = voice

The wordmark answers "are you looking?"; the mark answers "who is speaking?". Two variables, never more.

Nothing decorative moves

Only the wordmark, headlines, and the mark animate. Body text, rules, and layout stay still — the institution doesn't fidget.
