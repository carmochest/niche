01 — One of many

The mark is not drawn. It assembles.

No single dot matters; together they're unmistakable. The crowd scatters when you push through it — and always finds its shape again. That's the institution.

1
dot = one student

The unit of the brand is a person, not a pixel. Headcount literally shapes the mark — a bigger cohort is a denser mark.

3
shapes to gather around

The crowd forms the blob, the circle, or the triangle — the same shape language as the rest of the system.

0
outlines

The swarm never gets a containing stroke. Its edge is fuzzy because real communities are — held together by pull, not fences.

03 — Rules

How the swarm behaves.

Alive on screens, settling in print

Digital surfaces run the simulation. Print uses a frame exported from it — never a perfectly regular dot grid, which would kill the life.

The crowd has one color

All dots share the state color. A multi-colored swarm reads as confetti; a single-colored one reads as intention.

Wordmark stays solid

NICHE itself is always set in solid Libre Franklin 900. Only the mark is a crowd — the name is the building they're in.
