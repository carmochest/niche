# NICHE Institute — Student Portal (UX mockup)

A clickable single-file UX/UI mockup for a student learning portal at the NICHE Institute.

## What's in here
- `index.html` — the entire mockup (HTML + CSS + JS in one file)

## Run locally
Just open `index.html` in any modern browser. No build step.

## Tabs
- **Chat** — Claude-style "ask anything" interface
- **Coursework** — live online classes (join video call), in-person classes (QR check-in), search, materials, submissions upload
- **Progress** — per-course journey with milestones, instructor feedback, AI portfolio generation, link-to-job-portal

## Right rail (always visible)
- Announcements
- Reminders
- Job portal (shows linked portfolio + matched opportunities)

## Other
- Day / night theme toggle (top right; respects system preference on first load)
- Account drawer (click the avatar) — enrolment, tuition, settings, privacy

Status: design / UX prototype.
